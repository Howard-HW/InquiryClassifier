프로그램 소개

이 프로그램은 문의사항을 자동으로 분류하는 머신러닝 기반의 텍스트 분류 프로그램입니다.

주요 기능

  • 텍스트 기반 문의사항 자동 분류
  • 분류 결과 검수 및 수정
  • 수정된 결과를 반영한 모델 재학습
  • 사용자 정의 전처리 규칙 관리
  • 분류 규칙 관리

사용된 기술

머신러닝 모델:

  • TF-IDF (Term Frequency-Inverse Document Frequency)
    - 텍스트를 수치화하여 각 단어의 중요도를 계산
    - 문서에서 특정 단어의 중요성을 평가
  • 나이브 베이즈 분류기 (Multinomial Naive Bayes)
    - 텍스트 분류에 특화된 확률 기반 알고리즘
    - 새로운 데이터에 대한 빠른 학습과 예측

작동 방식

  1. 문의사항 텍스트 데이터 입력
  2. 사용자 정의 전처리 규칙 적용
  3. TF-IDF를 통한 텍스트 벡터화
  4. 나이브 베이즈 모델을 통한 분류
  5. 사용자 검수 및 수정
  6. 수정된 결과를 반영한 모델 재학습
